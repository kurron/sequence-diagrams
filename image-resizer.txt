title Image Resizer Sequence

WEBDAM->Web Hook: Asset change notification
Web Hook->SQS: Asset change event
SQS->Asset Copier: Asset change event
Asset Copier->WEBDAM: 1024x1024 image
Asset Copier->S3: 1024x1024 image
API Client->CloudFront: Image reqeust with size parameters

alt Client specified the 1024x1024 image
    CloudFront->S3: fetch specified image
else Client specifed a non-1024x1024 image
    CloudFront->Image Resizer: fetch specified image
end
